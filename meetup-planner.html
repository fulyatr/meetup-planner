Quick setup — takes 2 minutes

This app saves everyone's availability to a free cloud database so the whole group shares one permanent link.

1
Go to jsonbin.io and create a free account.
2
Once logged in, click API Keys in the top menu → click + ADD API KEY → name it anything → copy the key that appears.
3
Paste your API key below and click Create. A permanent link will be generated — share it with your friends and everyone uses the same one.
